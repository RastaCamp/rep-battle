## How AI / devs should use this file
- Edit JSON for gameplay numbers (passRate, failRate, timingMultiplier).
- Edit bio/tagline here or in JSON for flavor text.
- Add new entries by copying the template at the bottom.
- colorArgb is Flutter Color value (ARGB32). NPC picks own color in JSON.

## Bark lines (in-game speech)
All arrays live in assets/data/npc_profiles.json (or run tool/build_npc_json.py to merge quote batches).

| Field | When it plays |
|-------|----------------|
| quotesVictory | NPC will PASS a normal card (~34% chance) |
| quotesForfeit | NPC will FAIL |
| quotesModified | NPC will use MODIFIED |
| quotesClutch | NPC PASS on tough cards (8+ reps, Q/K/Joker) |
| quotesStartMatch | Match begins (~52% per NPC, staggered) |
| quotesLegendary | Rare ~7% override on any bark |
| quotesTeamUp | King / group challenges with 3+ players |
| quotesRespect | After human PASS/MODIFIED on tough card or combo 3+ (~42%) |
| quotesPain | NPC struggling mid-turn (modified / tough cards) |
| quotesHumiliated | NPC FAIL (~58% instead of standard forfeit line) |

Edit tool/npc_quotes_respect_pain.py for respect/pain batches, then: python tool/build_npc_json.py
Quote-only patches: tool/npc_quotes_batch5_extra.py → python tool/merge_npc_quotes.py

## Quick stats legend
- passRate: chance to PASS a card (0.0–1.0)
- modifiedRate: chance to use MODIFIED instead of full PASS when struggling
- failRate: extra fail weight (lower is better for player experience)
- timingMultiplier: scales (reps + 3) second turn timer

## Adding more NPCs
1. Add full profile object to `assets/data/npc_profiles.json` **or** drop a batch JSON in `tool/` and run `python tool/merge_new_profiles.py tool/your_batch.json`
2. Add portrait: `assets/images/npcs/{id}.png`
3. Update this file with a short summary
